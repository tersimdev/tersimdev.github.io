Gumball

This is a OpenGL and C++ project created for school, focusing on physics.
This is a 2-player pinball game where players fling balls towards the portal (orange/blue area) and try to score into the opponents goal (brown area).
The collisions are accurately detected and resolved using physics equations (ball-ball, ball-flipper, ball-wall, box trigger).
